# Playwright Google Search Automation

This repository contains a Playwright automation script that searches Google for "what is the future of SDET role in IT" and runs on GitHub Actions.

## Features
- Automated Google search using Playwright
- Cross-browser testing (Chromium, Firefox, WebKit)
- GitHub Actions integration
- Screenshot capture
- Search result validation
